- Для каждой фигуры находим все возможные позиции размещения на текущей доске
- Выбираем первую фигуру случайным образом из тех, которые можно разместить
- Размещаем первую фигуру на временной доске
- Ищем фигуры, которые можно разместить после первой
- Выбираем вторую фигуру случайным образом
- Размещаем вторую фигуру на временной доске
- Ищем фигуры для третьего размещения
-Выбираем третью фигуру случайным образом

- Если на каком-то этапе не удается найти подходящие фигуры, возвращаются случайные фигуры
- Каждая следующая фигура выбирается с учетом уже размещенных фигур
- Все размещения рандомизируются, если есть несколько вариантов
- Первая фигура гарантированно может быть размещена
- Вторая и третья фигуры выбираются так, чтобы их можно было разместить после первой
